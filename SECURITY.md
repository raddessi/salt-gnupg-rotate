# Security Policy

## Supported Versions

Here is a list of the release versions that should and should not be receiving
security updates:

| Version | Supported |
| ------- | --------- |
| 1.x.x   | Yes       |
| 0.x.x   | No        |

## Reporting a Vulnerability

If you find a vulnerability, please submit a bug report via the GitHub issue
tracker. Thanks!
